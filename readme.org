* 汇编及体系结构
** 8086 体系结构
*** 概述
8086 是 16 位的体系结构，采用小端的存储方式
*** 寄存器

|--------------+------+---------------------|

| 指针寄存器   | BP   | Base Pointer        |

| 段寄存器     | CS   | Code Segment        |

| 段寄存器     | SS   | Stack Segment       |
| 段寄存器     | ES   | Extra Segment       |
